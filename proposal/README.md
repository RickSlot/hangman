#HANGMAN PROPOSAL

The implementation of hangman in iOS. You can also add words to keep the replay value higher.

###Basic Features
- Play the game
- Pick a letter
- Get feedback on the chosen letter
- Change the hangman image
- Add new words

###Spare time features
- Evil hangman
- Add sounds

###Frameworks and other Technologies
- Objective-c
- Xcode
- Storyboards
